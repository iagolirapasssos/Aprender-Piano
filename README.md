# Piano Virtual de 88 Teclas com Tutorial

Este é um projeto de um piano virtual de 88 teclas com funcionalidades de tutorial e criação de músicas personalizadas. As músicas podem ser tocadas, praticadas e novas músicas podem ser adicionadas diretamente no navegador usando o LocalStorage.

## Funcionalidades

- **Piano Virtual**: Tocar notas usando o teclado do computador ou clicando nas teclas do piano.
- **Tutorial**: Aprender músicas seguindo as notas destacadas na partitura e no teclado.
- **Criação de Músicas**: Adicionar novas músicas personalizadas diretamente no navegador.
